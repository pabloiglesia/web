Click n'Print (August 2017 - Present)

At Click n'Print I have the opportunity to be part of a start up, with agile work methodology and a flexible way of working that allows us to easily adapt to changes in the project. I have been able to continue learning and applying all my public cloud knowledges learned during my previous experience in Room714 since I have developed the web application on which the company model is based. The developed on my own the web application is based on the following principles:

The developed web application is based on the following principles:

  • Robustness: Application developed on Python, using the Django framework.
  • Availability and Scalability: Built on top of the AWS public cloud, the server is ready for performing on-demand auto-scale to deliver 100% availability. AWS S3 is used to store all the documents of the page to ensure its persistance in case of system failure, and AWS lambdas are triggered periodycally to assure that the order's printing time commitmet is met.
  • Security: Through AWS's own protection systems and those of third parties on the server. In addition to good practices in programming.
  • Traceability: Using AWS Cloudwatch log system to centralize the storage and monitoring of the logs produced by our servers.

  • Developed on IBM Symphony Platform
The project is detailed in detail in my Final Degree Project. A work carried out with the full consent of the bank with which I obtained a rating of 9.5 out of 10.
